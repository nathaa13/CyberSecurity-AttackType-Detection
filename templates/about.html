CYBERVAULT

CyberVault is a machine-learning-powered web application designed to detect and classify cyber-attacks based on real-time network traffic data. Built as part of the Cyber Security Attack Type Detection project, this application provides a user-friendly interface where security analysts and IT professionals can analyze network activity and identify potential threats.
